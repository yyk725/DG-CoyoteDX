# DG-CoyoteDX

将你的 maimai DX 连接到郊狼上

Connect your maimai DX to DG-Lab Coyote E-Stim Powerbox

## 功能

- 目前只是一个功能原型，"It just works"。
- 展示一个二维码，郊狼 App 通过 Socket 连接到游戏中
- 当 1P 出现 Miss 判定，生成 100% 的脉冲强度；Good 判定生成 50% 的脉冲强度；Great 生成 25% 的脉冲强度。输出的强度需要用户手动调整。

## 计划

- 增加更多自定义选项，如更多波形、强度自动变强等
- 2P 模式和 1P/2P 互换模式
